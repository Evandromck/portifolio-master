Conhecimentos

HTML

O HTML5 não é mais apenas um construtor de websites. É um construtor de aplicações inteiro por conta própria. Quase tudo o que você pode fazer com estes sistemas pode ser feito de forma mais simples e eficiente em HTML5.

CSS

É possível realizar mudanças pontuais em um componente de um arquivo da web sem comprometer o resto do conteúdo. Isso pode ser feito de forma simples, apenas referenciando elementos no CSS3

Javascript

Adicione interatividade ao seu site, JavaScript é a linguagem de programação cheia de recursos que quando aplicada em um documento HTML, pode fornecer interatividade dinâmica em sites.

Python

Python traz características que possibilitam escrever o mesmo requisito em menos linhas de código que o necessário, utilizado em aplicações que lidam com processamento de texto, machine learning e recomendação de conteúdo.

Power BI

Fornecer visualizações interativas e recursos de business intelligence com uma interface simples para que os usuários finais criem os seus próprios relatórios e dashboards.

Git e GitHub

Plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele permite que programadores, utilitários ou qualquer usuário.
